# Kaggle Competitions | DADS - AI/ML 

This repository contains my Kaggle competition projects, focused on applying Data Analytics, Data Science, and Machine Learning concepts to real-world datasets.

## Focus Areas :
- Data preprocessing and cleaning
- Exploratory data analysis (EDA)
- Feature engineering
- Machine learning model building
- Model evaluation and improvement

## Structure :
Each competition is organized in its own folder and includes:
- Notebook (analysis + model)
- Submission file
- Project-specific README

## Tech Stack :
- Python
- Pandas, NumPy
- Scikit-learn
- Kaggle Notebooks

## Objective :
To build practical, end-to-end machine learning solutions and continuously improve problem-solving skills in Data Analytics and AI/ML.
