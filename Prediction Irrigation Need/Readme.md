# Predicting Irrigation Need 

## Problem :
Predict irrigation requirement (Low, Medium, High) using environmental and soil-related data.

---

## Approach :

### Data Processing
- Cleaned dataset
- Handled categorical variables using one-hot encoding
- Prepared train and test datasets consistently

### Feature Engineering
- Created basic interaction features to improve model understanding

### Model
- Random Forest Classifier
- Used class weighting to handle imbalance

---

## Result :
- Kaggle Public Score: **0.96188**

---

## Key Learnings :
- Importance of feature engineering
- Handling categorical data correctly
- Impact of class imbalance on model performance
- End-to-end ML pipeline execution

---

- Python
- Pandas, NumPy
- Scikit-learn
